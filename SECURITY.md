# Security Policy

## Supported Versions

We release security updates for the current major version. Check [Releases](https://github.com/alex-morozov84/AMCore/releases) for supported versions.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately:

1. **Do not** open a public issue.
2. Email **mail@alex-morozov.com** with:
   - A short description of the vulnerability
   - Steps to reproduce
   - Impact and possible mitigations (if known)
3. You will receive an acknowledgment within a few days. We will keep you updated on the status of the fix and coordinate disclosure if needed.

We appreciate responsible disclosure and will credit reporters (unless you prefer to stay anonymous) when the issue is fixed.
