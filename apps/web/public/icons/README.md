# PWA Icons

Place PWA icons here:

- `icon-192x192.png` — 192x192 pixels
- `icon-512x512.png` — 512x512 pixels

Icons should be square PNG files with the app logo.

## Generation

Use a favicon generator like:

- https://realfavicongenerator.net/
- https://favicon.io/

Or create manually with design tools.
